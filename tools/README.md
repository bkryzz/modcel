# MODCEL Tools

This directory contains helper tools for preparing MODCEL input files.

The tools are public input-authoring utilities. They do not include the
simulation engine, private model files, or project-specific biological
parameters.

## Tools

- `model-input-editor`: GUI utilities for editing structured MODCEL input files
  and cell phenotype/interactions files.
- `metabolic-interpreter`: GUI prototype for building small metabolic ODE
  systems and exporting a Fortran90 `computeRates` routine.
